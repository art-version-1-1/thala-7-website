# thala-7-website

A simple demo shoe shopping website built with HTML and CSS.

## Features

- Modern landing page layout with hero content and product imagery.
- Top navigation bar with:
  - branded logo styling
  - navigation links for Home, About, Brands, More, and Contact
  - search box with custom rounded styling and focus state
  - cart icon area and login button
- Prominent call-to-action button for shopping (`Shop Now`).
- Styled hero text, paragraph, and image section using CSS variables for colors.
- `shoe-page.html` placeholder product page linked from the landing page.
- Simple footer styling for consistent visual balance.

## Files

- `index.html` — main landing page.
- `shoe-page.html` — product page placeholder.
- `style.css` — site styling and layout rules.
- `images/` — image assets used by the site.

## Notes

- The `Shop Now` button is styled as a clickable anchor link.
- The search input uses custom border colors and rounded corners.
- The site is designed for a clean desktop-style UI with easy customization.
